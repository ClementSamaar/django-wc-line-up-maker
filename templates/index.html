Welcome to World Cup Line-Up Maker

Here, you can create your own lines-ups of the 2022 World Cup teams. For now there are only 8 teams available. For your lines-ups you can use the basic formations that we've created but also create your own formations templates. Also, the lines-ups that you are going to create are public, they can be seen by any visitor of the website.

How to create a formation and a line-up

As mentioned before, here, you are able to create formations and lines-ups. To create both of these, you will first create the overall definition of the formation/line-up. For a formation, it means to define the number of players at each game area (attack, midfield, defense) and also an optional additionnal information about the formation. For a line-up you'll have to define its name, the squad and the formation that you want to use. The second step is to create the template. For a formation, a template allows the user to choose the positions of the formation. For a line-up, you have to define each player position based on the formation you chose. If you haven't created a template for a formation or a line-up you can create or modifying it anytime by going on its detailed page.
